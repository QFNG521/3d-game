# 坦克大战 (Tank Battle)

经典坦克大战网页版游戏，使用纯 HTML/CSS/JavaScript 开发。

## 🎮 游戏玩法

- 控制你的绿色坦克消灭所有红色敌方坦克即可过关
- 击毁砖墙可以开辟新路径，但钢铁墙壁无法破坏
- 穿过树丛可以隐藏行踪，水域会阻挡坦克通行
- 每关敌人数量递增，AI 也会越来越聪明

## 🕹️ 操作说明

| 按键 | 操作 |
|------|------|
| `W A S D` 或 `↑ ↓ ← →` | 移动坦克 |
| `空格键` 或 `J` | 发射子弹 |
| `P` | 暂停/继续 |

## ✨ 特性

- 🎯 多关卡设计（3种地图布局循环，难度递增）
- 🧱 4种地形：砖墙（可破坏）、钢铁（不可破坏）、水域（阻挡通行）、树丛（视野遮挡）
- 🤖 敌人 AI：随机移动 + 追踪玩家
- 💥 爆炸特效
- ❤️ 生命系统（3条命）
- 🏆 计分系统
- 🛡️ 重生无敌时间
- 📱 响应式布局

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

或使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器访问 `http://localhost:8000`
